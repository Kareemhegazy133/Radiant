# Radiant Game Engine (WIP)

Powerful and versatile game engine designed for cross-platform game development.

## Highlights

- **2D Rendering:** High-performance rendering for UI, particles, and sprites.
  
- **3D Rendering:** Physically-based rendering for stunning visual experiences.

- **Cross-Platform Support:** Develop on Mac, Linux, Android, and iOS.

- **Native Rendering API:** Seamless integration with DirectX, Vulkan, and Metal.

- **Viewer & Editor:** Fully featured applications for efficient workflow.

- **Scripting:** Fully scripted interaction and behavior.

- **Physics Engine:** Integrated 2D and 3D physics for realistic simulations.

- **Procedural Generation:** Create dynamic worlds with procedural terrain and world generation.

- **AI:** Implement intelligent NPCs with built-in AI support.

- **Audio System:** Comprehensive system for immersive soundscapes and effects.
